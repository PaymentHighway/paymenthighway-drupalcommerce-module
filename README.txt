COMMERCE PAYTTRAIL README
===================================
This module integrates Paymenthighway payment method with Drupal Commerce.

INSTALLATION
============
- Enable the module as usual.
- You will probably need to enable more of the Commerce modules to make your
  shopping cart work.
  
CONFIGURATION
=============
Configuration page is located at
admin/commerce/config/payment-methods/manage/commerce_payment_commerce_paymenthighway
